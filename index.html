Was ist CSS

Es steht für Cascading Style Sheets
Mann benutzt es um Websites zu designen
und die HTML Tags zu stylen.

Was kann CSS alles machen ?
Du kannst :
-Größe von Elementen ändern
-Position von Elementen bestimmen
-Layout für die Website machen (Grid,Flex)
-Elemente Farbe geben
-Responsive Webdesign
(diese Website ist auch Responsive)
-Keyframes benutzen um animation zu machen
-Und noch mehr ...

-Wie erstelle ich eine CSS Dokument

    Es gibt drei Wege CSS zu benutzen
    Erster Weg :
    Erstelle eine Dokument nur für CSS
    Du müsst immer als .css stehen haben alles davor kannst du ändern.
    Ich würde dir auch empfehlen CSS Datei in den gleichen Ordner
    zu haben wie die HTML Datei ist

-So verbindest du CSS Dokument mit dem HTML

    Schreibe im des HTML Dokuments das Element Link und drücke darauf
    Rel: legt fest für was das Dokument gebraucht wird.
    Im diesen Fall brauchen wir Stylesheet um CSS zu benutzen.

    Href: sagt wo das Dokument zu finden ist. Im diesen Fall ist das HTML Dokument im gleichen Ordner wie das CSS Dokument
